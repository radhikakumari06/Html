# My First Web Project - Gradient Hero Website

Hi there! This is one of my early web development projects that I built while learning the basics of frontend development. It is a clean, modern single-page hero section with a beautiful gradient background and some cool vector graphics.

## 🔗 Live Link
You can check out my live website here:
👉 **https://radhikakumari06.github.io/Html/**

## ✨ Key Features
- *Smooth Scrolling Navigation:* If you click on Home, About, or Services in the top menu, the page smoothly scrolls down to that exact section without reloading.
- *Modern UI Design:* Features a stylish dual-tone blue and purple gradient background that gives it a fresh look.
- *Responsive Layout:* The design automatically adjusts and looks clean on laptop and desktop screens.
- *Single Page Flow:* Everything is structured neatly on a single page for a seamless user experience.

## 🛠️ Technologies Used
- *HTML5:* Used to build the main structure, headings, and different sections of the page.
- *CSS3:* Used for the gradient background styling, clean layouts, font settings, and the smooth scrolling effect (scroll-behavior: smooth).

## 📂 Project Structure
- index.html - The main file containing both the HTML structure and embedded CSS styling.
- Ai service.png, hello.png, img.jpg - Vector images used to decorate the hero sections.
